# JAVA 
## 클래스와 데이터

- 클래스와 인스턴스
    - 클래스는 설계도
    - 인스턴스는 설계도로 만들어진 실체
        - 클래스로 메모리 공간을 잡아 만든 실체를 객체라고 한다.
- 객체와 인스턴스의 차이
    - 객체
        - 클래스에서 정의한 속성과 기능을 가진 실체
    - 인스턴스
        - 특정 클래스로부터 생성된 객체
        - 보통 객체를 특정 클래스의 인스턴스라고 언급할 때 사용한다.
- ***자바는 대입할 때 복사해서 대입한다.***
- iter 단축키
    - foreach 문 단축
